## 📄 Project Overview

This project is a fun, creative, and engaging single-page personal website that showcases:
- A themed introduction to the Chelsea football community.
- Personal profile and interests of Oyelola Oluwasegun.
- A welcoming message for fellow Chelsea fans with football banter humor.
- A professional goal of becoming a frontend developer in the football media industry.
- A fan sign-up form for Chelsea enthusiasts.
- Contact information including phone, email, and GitHub.

## ✨ Features

- ⚽ **Chelsea Football Theme** with a blue/white color palette
- 🖼️ Shadowed Chelsea logo background in the header
- 💬 Fun and witty welcome message
- 🧑‍💻 Profile section with image upload and hobbies
- 🥅 Personal goal as a frontend web developer for sports media
- 📝 Sign-up form for new Chelsea fans
- 📱 Fully responsive layout for mobile and desktop
- 🎨 Clean typography using [Poppins](https://fonts.google.com/specimen/Poppins)
